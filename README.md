# debug_map
IntelliJ IDEA Plugin `Debug Map`. 

Manage breakpoints and bookmarks by topics, with AI-powered automatic placement to keep every debug flow organized and persistent.

- AI integration: let AI automatically place breakpoints and bookmarks at key code points during code review, writing, or any other workflow — helping you trace, review, and persist long call chains without missing a step
- Activate a topic to show only its breakpoints — inactive topics are persisted in the background without interfering with other debug flows
- Organize breakpoints and bookmarks into named topics with a dedicated management panel
Import and export topics for sharing or reuse across projects
